# Bitcoin Market Sentiment vs Trader Performance Analysis

## 📌 Project Overview

This project analyzes the relationship between **Bitcoin Market Sentiment (Fear & Greed Index)** and **trader performance** using historical trading data from Hyperliquid. The goal is to identify patterns between market sentiment and trading outcomes, providing insights that can support smarter trading decisions.


## 📊 Datasets Used

### 1. Bitcoin Market Sentiment Dataset

- Date
- Fear & Greed Value
- Classification (Extreme Fear, Fear, Neutral, Greed, Extreme Greed)

### 2. Hyperliquid Historical Trader Data

- Account
- Coin
- Execution Price
- Size (USD & Tokens)
- Side (Buy/Sell)
- Timestamp
- Start Position
- Direction
- Closed PnL
- Fee
- Transaction Details


## 🎯 Objectives

- Clean and preprocess both datasets.
- Merge trader data with market sentiment data.
- Analyze trader performance under different market sentiments.
- Discover hidden patterns using Exploratory Data Analysis (EDA).
- Generate actionable insights for trading strategies.
